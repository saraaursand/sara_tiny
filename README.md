# visual_wake_word (sara_tiny)

This repository contains code and data for a Visual Wake Word project (retraining and models).

Quick notes:

- Large model files and generated artifacts are excluded via `.gitignore` (e.g. `*.h5`, `*.tflite`).
- To push this repo to a remote, add a remote with `git remote add origin <url>` and `git push -u origin main`.

If you want your commits to show your user and email globally, set them with:

```powershell
git config --global user.name "Your Name"
git config --global user.email "you@example.com"
```

Or set them locally in this repository if you prefer:

```powershell
git config user.name "Sara"
git config user.email "sara@example.com"
```

Enjoy working on your project!
