# Tiny-MobileNet (sara_tiny)

This repository contains code for pre-training, re-training and testing of the Tiny-MobileNet model.

- Re-trained models are are excluded via `.gitignore` (e.g. `*.h5`, `*.tflite`).
- Large datasets used during pre-training and re-trianing are excluedd via `gitignore`(e.g. `COCO2014`, `MNIST`)
